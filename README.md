# SellUsedStuff
🚛 realtime database와 storage를 이용한 중고 거래 앱 🚛

## 개발환경

* OS : windows 10 pro
* IDE : Android Studio
* Language : Kotlin

## 사용한 것

* RecyclerView
* ViewBinding
* BottomNavigationView
* Firebase Storage
* Firebase Realtime Database
* Firebase Auth
